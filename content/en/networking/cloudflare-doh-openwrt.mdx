# Cloudflare DNS over HTTPS (DoH) on OpenWrt

## Install packages

```bash
opkg update
opkg install https-dns-proxy luci-app-https-dns-proxy
```

## Configure

- LuCI → Services → DNS over HTTPS
- Provider: Cloudflare (1.1.1.1)
- Enable and Apply

Ensure DHCP hands out router as DNS to clients.
