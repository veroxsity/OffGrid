# qBittorrent with Docker

```yaml
services:
  qbittorrent:
    image: lscr.io/linuxserver/qbittorrent:latest
    environment:
      - PUID=1000
      - PGID=1000
      - WEBUI_PORT=8081
    ports:
      - 6881:6881
      - 6881:6881/udp
      - 8081:8081
    volumes:
      - ./config:/config
      - /srv/downloads:/downloads
    restart: unless-stopped
```

Open `http://SERVER_IP:8081` (default admin/adminadmin, change immediately).
