# Meilisearch with Docker

```yaml
services:
  meilisearch:
    image: getmeili/meilisearch:latest
    environment:
      MEILI_MASTER_KEY: change-me
    ports:
      - "7700:7700"
    volumes:
      - ./data:/meili_data
    restart: unless-stopped
```

Open `http://SERVER_IP:7700` and set an API key for clients.
