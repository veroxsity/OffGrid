# Radarr with Docker

```yaml
services:
  radarr:
    image: lscr.io/linuxserver/radarr:latest
    environment:
      - PUID=1000
      - PGID=1000
      - TZ=Europe/London
    ports:
      - 7878:7878
    volumes:
      - ./config:/config
      - /srv/media/movies:/movies
      - /srv/downloads:/downloads
    restart: unless-stopped
```

Open `http://SERVER_IP:7878`.
