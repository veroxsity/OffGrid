# Forgejo (Git hosting) with Docker

```yaml
services:
  forgejo:
    image: codeberg.org/forgejo/forgejo:latest
    user: "1000:1000"
    ports:
      - "3001:3000"
      - "223:22"
    volumes:
      - ./data:/data
    restart: unless-stopped
```

Open `http://SERVER_IP:3001`.
