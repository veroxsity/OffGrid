# Vaultwarden (Bitwarden) with Docker

```yaml
services:
  vaultwarden:
    image: vaultwarden/server:latest
    container_name: vaultwarden
    environment:
      - ADMIN_TOKEN=change-me
    volumes:
      - ./data:/data
    ports:
      - "8085:80"
    restart: unless-stopped
```

Reverse proxy with HTTPS strongly recommended.
