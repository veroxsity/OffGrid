# n8n (Automation) with Docker

```yaml
services:
  n8n:
    image: n8nio/n8n:latest
    environment:
      - N8N_BASIC_AUTH_ACTIVE=true
      - N8N_BASIC_AUTH_USER=admin
      - N8N_BASIC_AUTH_PASSWORD=change-me
    ports:
      - "5678:5678"
    volumes:
      - ./data:/home/node/.n8n
    restart: unless-stopped
```

Open `http://SERVER_IP:5678`.
