# Portainer (Docker UI) with Docker

Portainer lets you manage Docker visually.

## Deploy

```bash
mkdir -p ~/portainer && cd ~/portainer
cat > docker-compose.yaml <<'YAML'
services:
  portainer:
    image: portainer/portainer-ce:latest
    container_name: portainer
    ports:
      - "9443:9443"
    volumes:
      - /var/run/docker.sock:/var/run/docker.sock
      - ./data:/data
    restart: unless-stopped
YAML

docker compose up -d
```

- Use Stacks to deploy Compose files.
