# Gitea (Git hosting) with Docker

```yaml
services:
  gitea:
    image: gitea/gitea:latest
    user: "1000:1000"
    ports:
      - "3000:3000"
      - "222:22"
    volumes:
      - ./data:/data
    restart: unless-stopped
```

Visit `http://SERVER_IP:3000` to initialize.
