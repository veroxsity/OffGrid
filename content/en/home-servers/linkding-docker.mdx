# Linkding (Bookmarks) with Docker

```yaml
services:
  linkding:
    image: sissbruecker/linkding:latest
    ports:
      - "9090:9090"
    volumes:
      - ./data:/etc/linkding/data
    restart: unless-stopped
```

Login at `http://SERVER_IP:9090` (default admin created on first run).
