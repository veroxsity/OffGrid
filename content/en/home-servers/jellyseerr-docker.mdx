# Jellyseerr (Media Requests) with Docker

```yaml
services:
  jellyseerr:
    image: fallenbagel/jellyseerr:latest
    ports:
      - 5055:5055
    volumes:
      - ./config:/app/config
    restart: unless-stopped
```

Open `http://SERVER_IP:5055` and connect to Radarr/Sonarr.
