# Prowlarr with Docker

```yaml
services:
  prowlarr:
    image: lscr.io/linuxserver/prowlarr:latest
    environment:
      - PUID=1000
      - PGID=1000
      - TZ=Europe/London
    ports:
      - 9696:9696
    volumes:
      - ./config:/config
    restart: unless-stopped
```

Configure indexers and link to Sonarr/Radarr.
