# Navidrome (Self-hosted Music) with Docker

```yaml
services:
  navidrome:
    image: deluan/navidrome:latest
    user: "1000:1000"
    ports:
      - "4533:4533"
    volumes:
      - ./data:/data
      - /srv/music:/music:ro
    environment:
      - ND_LOGLEVEL=info
      - ND_SCANSCHEDULE=1h
    restart: unless-stopped
```

Open `http://SERVER_IP:4533` → create admin → scan /music.
