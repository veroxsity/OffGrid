# Uptime Kuma (Self-hosted Status Page) with Docker

Simple monitoring with notifications.

## Deploy

```bash
mkdir -p ~/uptime-kuma && cd ~/uptime-kuma
cat > docker-compose.yaml <<'YAML'
services:
  uptime-kuma:
    image: louislam/uptime-kuma:latest
    container_name: uptime-kuma
    volumes:
      - ./data:/app/data
    ports:
      - "3001:3001"
    restart: unless-stopped
YAML

docker compose up -d
```

Open `http://SERVER_IP:3001` and create an admin account.

Add monitors, set notifications (Telegram, Discord, Email, etc.).
