# FreshRSS with Docker

```yaml
services:
  freshrss:
    image: freshrss/freshrss:latest
    ports:
      - "8086:80"
    volumes:
      - ./data:/var/www/FreshRSS/data
      - ./extensions:/var/www/FreshRSS/extensions
    restart: unless-stopped
```
